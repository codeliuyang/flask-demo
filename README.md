## Common Questions
- ```__name__``` 代表什么意思？

```__name__``` 是当前模块名，当模块被直接运行时模块名为 ```__main__``` 。

- python中的模块怎么理解？

在Python中，一个.py文件就是一个模块，一般情况下，模块的名字就是文件名(不包括扩展名.py)。
全局变量__name__存放的就是模块的名字。
